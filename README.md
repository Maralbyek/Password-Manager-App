# Backend — Password Vault (Python)

This backend handles password storage and encryption.  
It uses a simple **file-based vault** with Base64 encoding.

##  Run the Backend

1. Navigate to backend folder:
2. Run the app:

This opens a **CLI menu** that lets you:
- Save passwords  
- View masked passwords  
- View unmasked passwords  
- Search by website  
- Generate strong passwords
